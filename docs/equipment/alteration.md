# 全改、下改石使用方法


:::caution
全改造石不易取得、請用在課金裝或唯一裝備上
:::


- 1.強化前用下改石改下去:把三屬性全部重選並重骰數值.這時有機率出我要的迴避/命中/速度,但數值可能不是頂配.

- 2.強化前用全改石改下去:把三屬性全部重選並重骰數值,這時有機率出我要的迴避/命中/速度,然後各屬性除原始屬性(可能不是頂配)外,還多給一次精煉數值加成,但各屬性仍然可以去鐵匠那精煉一次.

2.的情況是當你用全改改後前3條假設是你要的了，但你強化時的強化追加護卻跳爛比如攻擊+3，那你可能不滿意又要重骰，就又得再用全改去重骰去改

1.也是一樣，只是不像2那麽傷本。 所以也有人是等+20後再來用重骰，但如果是這樣，你可能就要下決心不管骰到啥詞條，我就是只單改第4條，才不會造成第1.2.3詞條的追加護損失

因為你強化後全改後再單改1.2.3條，就是同時損失全改+強化的2次追加護，而第4條是套用白裝機制，所以再單改你也完全沒任何追加護損失。
