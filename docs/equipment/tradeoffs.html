如何篩選裝備

武器篇​

雙手武器新手篇：長槍、弓箭、大劍、雙手斧、雙手錘​

備註

※ 給1、2章開荒玩家參考用，第三章以後的請往下看老手篇。

推薦程度：長槍 = 弓箭 = 雙手斧 > 大劍 > 雙手錘
建議詞條：攻擊 > 命中 > 會心 >> 其他
堪用等級的品質：3★白色以上
好用等級的品質：4★藍色以上（至少有兩條你想要的加護）

速成堪用雙手武器的製作方法​

  1. 找一把起始有攻擊力（固定值）詞條的雙手武器，星數不限，顏色不限
    ※ 如果是四星紫色等高級胚子，請看下方老手篇
  2. 如果是藍色的，可以在前兩條中選一條改為攻擊
  3. 強化到+10

雙手武器老手篇​

推薦程度：長槍 = 弓箭 = 雙手斧 > 大劍 > 雙手錘

備註

※ 如果有種族歧視需求，可以無視以上選擇有歧視效果的。

建議詞條：會心 = 攻擊 > 命中 >> 其他
堪用等級的品質：3★藍色以上
好用等級的品質：3★紫色／4★藍色以上（至少有兩條你想要的加護）

單手武器篇：單手劍、匕首、盾牌​

推薦程度：單手劍 = 匕首 >>> 斧頭 >>> 單手錘
建議詞條（武器）：攻擊力 = 會心 > 迴避 = 命中
建議詞條（盾牌）：會心 = 迴避 = 速度 >>> 其他（請根據你缺的數值選擇）
堪用等級的品質：3★藍色以上（至少有一條你想要的加護）
好用等級的品質：4★藍色以上（至少有兩條你想要的加護）

雙手法杖篇​

建議詞條（法師）：（會心） > 魔力 > 速度 > 神力 >>> 其他
建議詞條（僧侶）：（會心） > 速度 >>> 神力 >>> 其他
建議詞條（法僧雙修）：（會心） > 魔力 > 速度 > 神力 >>> 其他
堪用等級的品質：任意當前最好的雙手杖／第三章後1★白沙漠雙手杖
好用等級的品質：3★紫色／4★藍色以上（至少有兩條你想要的加護）

備註

※註：會心的部分請根據你是否是新時代法師決定
備註：非弓的武器不會出速度詞條，我估計這是文章作者舊版本沒更新，請自己找爛詞條打速度上去

沙漠雙手法杖的製作方法​

  1. 強化到+5，沒出（會心）／魔力就當合一狗糧
  2. +5出了（會心）／魔力以後，就可以繼續強化到+10
  3. +10的詞條如果不是（會心）／魔力，變造（會心）／魔力
    （當然如果你很有錢可以繼續嚴選第三條，不過太貴了不建議）
  4. 加上精煉以後，你就可以擁有一把約（+8~+15會心）／+10~20魔力的雙手法杖了

單手法杖篇​

建議詞條（法師）：（會心） > 魔力 > 速度 > 神力 >>> 其他
建議詞條（僧侶）：速度 > 神力 >>> 其他（雙速度或者三速度詞條為佳）
建議詞條（法僧雙修）：（會心） > 魔力 > 速度 > 神力 >>> 其他
堪用等級的品質：沒有堪用分級
好用等級的品質：3★紫色／4★藍色以上（至少有兩條你想要的加護）

備註

※註：會心的部分請根據你是否是新時代法師決定
※註：個人不推薦舊時代法師使用該武器。
備註：非弓的武器不會出速度詞條，我估計這是文章作者舊版本沒更新，請自己找爛詞條打速度上去

防具篇​

頭部、手部​

建議詞條（法師）：魔力 > 抵抗 = 魔防 >>> 其他
建議詞條（僧侶）：神力 > 抵抗 = 魔防 >>> 其他
建議詞條（近戰職）：攻擊 > 命中 > 抵抗 = 魔防 >>> 其他
堪用等級的品質：3★藍色以上（至少有一條魔力、神力或者攻擊力）
好用等級的品質：3★紫色／4★藍色以上（至少有兩條你想要的加護）

由於頭部與手部是屬於會出現攻擊、魔力、神力這三個主要屬性的部位，因為有沒有出現這三個對應的主屬性是關鍵的一點。

這兩個部位算是代差比較小的部位，再加上還需要嚴選上述的三詞條，所以對於裝備本身的白板要求沒那麼高，鐵裝等級以上即可使用。

如果是3★藍色、3★紫色上面有單有用詞（攻擊＆命中）這種場合，會建議把較沒用的第二條改成攻擊或命中，因為強化增強幅度較大，不過因為是過渡裝備，不建議使用太高級的變造、精煉石頭！

如果是4★藍色、4★紫色上面有單有用詞（攻擊＆命中）這種場合，會建議把較沒用的第二條改成攻擊或命中，因為強化增強幅度較大。

如果是3★、4★藍色上面有雙有用詞（攻擊＆命中）這種場合，建議會留到第三、第四條找比較差的改會心或者速度。

如果是3★、4★紫色上面有雙有用詞（攻擊＆命中）這種場合，會建議把較沒用的第三條改成攻擊或命中，因為強化增強幅度較大；如果你的攻擊跟命中夠用了也是可以考慮改成會心。

如果紫色上面有三個有用詞，那就恭喜你，可以把變造留給最後開出來的那一格了！
這種況通常建議會改成會心或者速度。

盾牌、身體​

建議詞條（法師）：（會心） = 速度 > 抵抗 = 魔防 >>> 其他
建議詞條（僧侶）：速度 > 抵抗 = 魔防 >>> 其他
建議詞條（近戰職）：會心 = 迴避 = 速度 > 抵抗 = 魔防 >>> 其他
堪用等級的品質：3★藍色以上（至少有一條會心、迴避或者速度）
好用等級的品質：3★紫色／4★藍色以上（至少有兩條你想要的加護）

由於盾牌與身體是屬於會出現會心、迴避、速度這三個主要屬性的部位。
請根據整體狀況，當下缺乏的數值進行變造與調整。

注意的是，盾牌與身體部位的防禦數值很高，是屬於代差較明顯的部位，如果已經到第三章的朋友建議至少從鋼裝以上再開始做。不過鐵裝如果有好的加護與高星數也是依然可以考慮。

如果是3★藍色、3★紫色上面有單有用詞（會心、迴避、速度）這種場合，會建議把較沒用的第二條改成迴避，因為強化增強幅度較大，不過因為是過渡裝備，不建議使用太高級的變造、精煉石頭！

如果是4★藍色、4★紫色上面有單有用詞（會心、迴避、速度）這種場合，會建議把較沒用的第二條改成迴避，因為強化增強幅度較大。

如果是3★、4★藍色上面有雙有用詞（會心、迴避、速度）這種場合，建議會留到第三、第四條找比較差的改會心或者速度。

如果是3★、4★紫色上面有雙有用詞（會心、迴避、速度）這種場合，會建議把較沒用的第三條改成迴避，因為強化增強幅度較大；如果你的迴避夠用了也是可以考慮改成會心或速度。

如果紫色上面有三個有用詞，那就恭喜你，可以把變造留給最後開出來的那一格了！
這種況通常建議會改成會心或者速度。

建議詞條（近戰職）：命中 = 迴避 = 速度 > 魔防 >>> 其他

戒指​

建議詞條（有生存需求時）：迴避 = 速度 = 抵抗 >>> 其他
建議詞條（法師）：（會心） = 魔力 = 速度 >>> 其他
建議詞條（僧侶）：速度 > 神力 = 抵抗 >>> 其他
建議詞條（近戰職）：會心 = 攻擊 = 速度 >>> 其他
堪用等級的品質：3★藍色以上（至少有兩條你想要的加護）
好用等級的品質：3★紫色／4★藍色以上（至少有兩條你想要的加護）

戒指的特性是會出現所有的加護，此外因為產量相對高，
所以堪用等級我認為要比其他部位要來的高一點。
大多數的情況下，飾品這個部位還是增加輸出為主，常見的配置就是增加會心、攻擊力與魔力。
但是如果有生存壓力，請不要懷疑戴上偏生存能力的飾品吧！

如果是3★、4★藍色上面有雙有用詞這種場合，建議會留到第三、第四條找比較差的改會心或者速度。

如果是3★、4★紫色上面有雙有用詞這種場合，會建議把較沒用的第三條改成攻擊、魔力、命中或迴避，因為強化增強幅度較大；如果你的迴避夠用了也是可以考慮改成會心或速度。

如果紫色上面有三個有用詞，那就恭喜你，可以把變造留給最後開出來的那一格了！
這種況通常建議會改成會心或者速度。
